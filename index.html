Video Game Design Project

Description

Our group had to create a video game that had to be able to teach people. We made a math game for ages 7+. In our game, players will be able to:

- Solve equations in different ways

- Earn coins to purchase cosmetics in order to make each players' experience unique to themselves


Design and Production Process:

1. Thought of different games that are enjoyed by all ages.

2. Researched well known games

3. Created mockups of the game

Reflection

After doing this project, I learned about:
- How to work with people
- Researching games that are played by all age ranges
- Think of different ways to create a game
